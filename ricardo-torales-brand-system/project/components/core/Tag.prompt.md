# Tag

Small pill/chip for skills, technologies, and keywords. Rounded-square on light surfaces, translucent pill on the dark hero.

```jsx
<Tag>PMBOK</Tag>
<Tag tone="onDark">Dynamics 365</Tag>
<Tag tone="accent">IA aplicada</Tag>
```

Tones: `solid` (slate chip) · `onDark` (translucent, for navy bands) · `accent` (sky-tinted).
