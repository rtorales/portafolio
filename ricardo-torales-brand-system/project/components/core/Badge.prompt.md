# Badge

Uppercase status/label pill. Use `success` + `dot` + `pulse` for "Activo"/availability states, `gold` for distinctions and scholarships, `accent` for date/period pills.

```jsx
<Badge tone="success" dot pulse>Activo</Badge>
<Badge tone="accent">2026 – Actualidad</Badge>
<Badge tone="gold">Becado</Badge>
```

Tones: `success` · `gold` · `accent` · `neutral`.
