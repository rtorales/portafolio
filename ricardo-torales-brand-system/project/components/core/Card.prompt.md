# Card

Base surface container: white, slate border, 12px radius. Use `interactive` for the hover-lift used on skill/cert cards, `accent` for the tinted highlight-box (left accent border) used to emphasize key statements.

```jsx
<Card interactive>…</Card>
<Card accent><p>Actualmente Gerente de Proyectos en Grupo Chacomer…</p></Card>
```
